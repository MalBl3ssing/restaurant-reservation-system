Smart Restaurant App

- Description

Smart Restaurant App is a restaurant website created for a school project using HTML, CSS, and JavaScript.

- The website allows users to:

View restaurants
View restaurant menus
Add food items to cart
Book tables
Login and register accounts
Access help and support pages.

-  Features
Home page with automatic image slider
Restaurant page
Food menu page
Shopping cart page
Booking page
Login and register pages
Help and support page
Navigation buttons between pages.

-  Technologies Used
.HTML
.CSS
.JavaScript
